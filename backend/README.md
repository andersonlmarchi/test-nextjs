# Gestão de Contratos de Crediário da Maria

## Descrição

Backend desenvolvido em NestJS para o Sistema de Gestão de Contratos de Crediário da Maria criado por Anderson © 2025

## Instalação

Necessário ter Node na versão 22 e NPM acima da 10

```bash
$ npm install
```

## Executar o projeto

```bash
# watch mode (com nodemon)
$ npm run start:dev
```

## Rodar testes

```bash
# unit tests
$ npm run test
```

## Documentação da API

A documentação da API estará disponível em:
```bash
http://localhost:3000/docs
```
