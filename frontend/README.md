# Gestão de Contratos de Crediário da Maria

## Descrição

Aplicação frontend construída em **Vue 3** + **Vite** para consumir a API do backend (NestJS).  
O sistema permite upload de arquivos JSON com contratos e visualização dos dados processados.

## Instalação

Necessário ter Node na versão 22 e NPM acima da 10

```bash
$ npm install
```

## Executar o projeto

```sh
npm run dev
```

## Integração com o Backend

O frontend se comunica com o backend (NestJS) rodando em:

```bash
http://localhost:5173
```

Certifique-se de que o backend esteja ativo antes de interagir com as funcionalidades.
O servidor de desenvolvimento estará disponível em `http://localhost:5173/` (padrão do Vite).
